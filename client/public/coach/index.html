Questions teachers are asking.

Do I need to know anything about AI or "prompt engineering" to use this?

No. The prompt is written. You paste it into any AI chat tool (ChatGPT, Claude, Gemini, Copilot), edit two lines — your standard and your differentiator — and run it. If you can send an email, you can run this prompt.

What grade levels does this work for?

Grades 3 through 12. The tiering logic is the same across grade bands; the examples in the PDF include a 5th grade science lesson and an 8th grade algebra review so you can see it working at both ends of the spectrum.

Which AI tool should I use?

Any major chat model works. ChatGPT (free or paid), Claude, Gemini, Copilot. The XML tag structure is what matters — it's how the model knows which part is the role, which is the task, which is the context, and which is the output format. Tag structure beats tool choice.

Is any of this going to get me in trouble with my district's AI policy?

The prompt is a tool. It generates tiered lesson drafts you review before using. It doesn't send student data anywhere. Follow your district's AI guidance the same way you'd follow it with any other generative tool — review outputs, redact student info, and use judgment. Ryan writes from inside a K-8 leadership role, and the PDF is built to be policy-safe when used responsibly.
